Eventure Sprint

A modern React and Vite front-end application focused on building
event-based interfaces with clean structure, responsive design, and
scalable component architecture.

------------------------------------------------------------------------

Overview

Eventure Sprint is a front-end project designed to practice modern web
development using React. It emphasizes reusable components, structured
styling, and integration with backend services.

------------------------------------------------------------------------

Features

-   Component-based React architecture
-   Responsive and mobile-friendly design
-   SCSS styling system
-   Client-side routing
-   API-ready setup using Axios
-   Clean and scalable project structure

------------------------------------------------------------------------

Tech Stack

-   React
-   Vite
-   JavaScript (ES6+)
-   SCSS
-   Axios
-   React Router DOM
-   React Syntax Highlighter
-   UUID

Installation

Clone the repository:

git clone https://github.com/hellsten/Eventure-Sprint.git cd
Eventure-Sprint

Install dependencies:

npm install

Run the development server:

npm run dev

------------------------------------------------------------------------

Backend

This project can be connected to a backend server available here:

https://github.com/hellsten/Eventure-Sprint-Server

Make sure the backend is running to enable full functionality.
